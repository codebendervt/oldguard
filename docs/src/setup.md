
# Setup

### Requirements 

The Deno runtime is required to run the old guard
Linux
`curl -fsSL https://deno.land/x/install/install.sh | sudo DENO_INSTALL=/usr/local sh`
Windows
`choco install deno`

To learn more about [Deno](https://deno.land/).
